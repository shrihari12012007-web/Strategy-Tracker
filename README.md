# Strategy Tracker

A personal productivity tool designed to track market strategies and daily coding routines. Built as a Progressive Web App (PWA) for desktop and mobile efficiency.

## Key Features
- **Progress Tracking:** Real-time circular SVG visualization of weekly productivity.
- **Data Persistence:** Uses LocalStorage to ensure your data remains saved across browser sessions.
- **Installable (PWA):** Can be installed as a native application on Windows and macOS.
- **Responsive UI:** Clean, modern interface styled with Tailwind CSS.

## Tech Stack
- **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS.
- **Architecture:** PWA (manifest.json).
- **Deployment:** Vercel (CI/CD integration).
